# Vantage Pulse — Marketing Workspace

This repository contains all marketing assets, content, and brand collateral for **Vantage Pulse** — an automated employee engagement pulse survey platform by Vantage Circle.

---

## Directory Structure

| Directory | Purpose |
|---|---|
| `/collaterals/` | PDF sales decks, case studies, brochures, guides, and product documentation |
| `/visual-context/` | Plain-text descriptions of charts, UI screenshots, and infographics extracted from PDFs |
| `/blogs/` | Blog catalog and URL list (43 posts across 8 topic categories) |
| `/vc-marketing/` | Product marketing knowledge base |

---

## Quick Reference

- **Brand voice:** Data-driven and authoritative, but human and direct. Never corporate-bland.
- **Primary case study:** AccessOne — 67% participation rate, eNPS 45
- **Key proof stat:** Gallup $483B–$605B/year cost of disengagement
- **Scale:** 700+ clients · 3.2M+ users · 100+ countries

## Working in This Repo

1. Read `/visual-context/` files first — they are the fastest way to access stats and product visuals without opening PDFs.
2. Check `/blogs/blog_catalog.md` for existing content before drafting new posts.
3. All marketing tasks must follow the brand voice and content principles in `CLAUDE.md`.

# Thumbnail Design Principles

Reference material for `youtube-thumbnail-designer`. CTR-tested patterns, color psychology, text density rules, mobile legibility, and pattern interrupt principles.

---

## 5 CTR-Tested Patterns for B2B / Edu YouTube

These are the archetypes that consistently show higher CTR in B2B and educational content. Note: what works for MrBeast-style consumer content (exaggerated faces, extreme contrast) doesn't always work for B2B — the audience self-selects differently.

### 1. Face + Big Text

Face (yours, guest's) occupies one side. 2-3 word text occupies the opposite side. Eye contact with camera. Facial expression conveys the emotion of the video's take.

**When it works:**
- Creator-led content (personal brand)
- Interview videos (guest's face + topic text)
- Opinion/commentary videos

**Example variant:**
- Face (left, 60% of frame) with skeptical expression
- Text (right): "Pipeline is fake"
- Brand color background

### 2. Stat Shock + Bold Number

The stat is the hero. Number fills 40-60% of the frame. Surrounding elements minimal.

**When it works:**
- Data-heavy videos
- Research reveals
- Contrarian takes backed by numbers

**Example variant:**
- "73%" in massive type (occupies 50% of frame)
- Subtext: "is fake"
- Brand color block behind

### 3. Before/After Split

Vertical or horizontal split. Left/top = "before" state. Right/bottom = "after." Usually desaturated/chaotic on "before," branded and ordered on "after."

**When it works:**
- Transformation content
- How-to videos where the change is visually obvious

**Example variant:**
- Left: chaotic CRM dashboard screenshot (desaturated)
- Right: clean, simplified dashboard (full brand color)
- Split arrow or "→" in the middle

### 4. Object + Annotation

A single object or product screenshot with one annotation or arrow pointing to a detail.

**When it works:**
- Product walkthroughs
- "The one feature you missed" style videos
- Tool comparisons

**Example variant:**
- Product screenshot crop (dashboard, specific UI)
- Bold arrow or circle highlighting one element
- 2-word callout: "This metric"

### 5. Meme-Format Reframe

Familiar meme structure (Drake pointing, two-button choice, distracted boyfriend) applied to B2B content. Done well = highly shareable. Done badly = cringe.

**When it works:**
- Contrarian takes
- "Stop doing X" videos
- Audience that recognizes the meme format (younger marketers, dev relations, product-focused)

**Risk:** Requires cultural fluency. If the meme feels forced, skip it.

## Face + Text Hierarchy

When using Pattern 1 (face + text):

- **Face placement:** Left or right, never center (centered faces look like a stock portrait).
- **Face size:** 40-60% of frame. Smaller = text dominates, larger = face dominates — both work.
- **Eye contact:** Direct gaze at camera outperforms averted gaze.
- **Expression:** Strong expression (surprise, skepticism, excitement) outperforms neutral. Don't overdo it — B2B audiences trust professionalism.
- **Text placement:** Opposite side from face. Same vertical plane as eye line (reader's eye flows face → text naturally).
- **Text size:** 80-120pt at full 1280x720 res for 4-word phrases. Scale up for fewer words.

## Color Psychology

Colors for thumbnails should be bolder and higher-saturation than on-brand web/print. The thumbnail has to fight for attention in a 300px-wide preview.

| Color tone | Emotion | Use when |
|---|---|---|
| Red | Urgency, danger, warning | Stat shock, contrarian, "stop doing X" |
| Yellow | Attention, warning, pop | Pattern interrupt — use sparingly |
| Orange | Warmth, energy, action | High-energy how-to, transformation |
| Navy | Authority, trust, stability | Research, framework, deep-dive |
| Green | Growth, success, money | Outcome-led videos, results content |
| Black | Premium, serious, editorial | Opinion, thought leadership |
| Brand primary | On-brand consistency | Channel-identity videos, tutorials |

**Avoid:**
- Muddy palettes (brown, olive, muted pastels) — invisible in feed
- Pure white backgrounds — disappear against YouTube's white/dark UI
- Gradient backgrounds — never high-contrast enough, read as "AI-generated"

## Text Density Rules

- **Max words on thumbnail:** 4. Ideal: 2-3.
- **Minimum font size at 1280x720:** 80pt for 4-word phrases, 100pt for 2-3 words, 120pt+ for 1-word phrases.
- **Contrast:** Test thumbnail in grayscale — if the text disappears, contrast is too low.
- **Font choice:** Sans-serif, bold weight, tight letter spacing. Examples: Inter Black, Montserrat Black, Bebas Neue, Anton.
- **All caps:** Generally outperforms title case for thumbnails — easier to read at small sizes.

---

## Mobile Legibility Test

Every thumbnail must pass the 300px test:

1. Generate the 1280x720 thumbnail
2. Scale it down to 300px wide (mobile preview size)
3. Does the text read?
4. Is the focal point clear?
5. Does it stand out against a white feed background?

If any answer is no, go back and increase contrast, size up text, or simplify the composition.

## Pattern Interrupt Principles

What breaks the YouTube home-feed grid:

1. **Bold, oversized single element.** The eye tracks "biggest thing on screen" first.
2. **Eye contact.** Human faces with direct gaze trigger immediate attention.
3. **Asymmetric composition.** Most thumbnails are centered; off-center composition interrupts the pattern.
4. **Color break.** If surrounding thumbnails are dark, go light. If they're blue-heavy (common in tech), go red/orange/yellow.
5. **Pattern vs. pattern.** Going extremely clean in a field of cluttered thumbnails works too — restraint can be pattern interrupt.

---

## Don't Do

- **No generic stock photography.** Instant signal of low effort.
- **No emoji in thumbnails.** Looks amateur for B2B, though works for consumer.
- **No logos as the dominant element.** The channel avatar already shows your logo.
- **No watermark strips at the bottom.** Steal thumbnail space from your hook.
- **No blurry or low-res imagery.** YouTube compresses; start with sharp source.
- **No small text that tries to "add depth."** Viewers won't zoom in.
